# Todo-list
* Скачать приложение используя команду:
> git clone   git@github.com:Yakovenko-Sergey3000/todo-list.git
* Создать базу данных в (**Postgresql**) c названием (**todolist**)
* В корневом каталого создать файл (**.env**) и задать переменные окружения
> PORT=
>
> DB_HOST=
>
> DB_USER=
>
> DB_PASSWORD=
>
> DB_DATABASE=
>
> DB_PORT=
>
> DB_SUPERUSER_PASS=

SUPERUSER - Пароль для пользователя регистрирующийся по дефолту.
* Пройти в каталог (**client**)
> cd todolist/client
* Установить пакеты с зависимостями
> npm i
* Зайти в папку (**server**)
> cd ../server
* Установить пакеты с зависимостями
> cd npm i
* Запустить проект в режиме разработки
> npm run dev
